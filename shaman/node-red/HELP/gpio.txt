1. apt-get install git-core
При возникновении каких-либо ошибок, тогда необходимо обновить вашу ОС:
sudo apt-get update
sudo apt-get upgrade

2. git clone https://github.com/zhaolei/WiringOP.git -b h3
В случае если вы уже клонировали файлы ранее, тогда необходимо просто обновить до последней версии:
cd WiringOP
git pull origin

3. Установка WiringOP/WiringPi
cd WiringOP
chmod +x ./build
sudo ./build

4. Проверка WiringOP/WiringPi
gpio -v
gpio readall

nano /etc/modules-load.d/modules.conf
тут вкл выкл фишшки

gpio_sunxi - гпио
w1-sunxi
w1-gpio
w1-therm - 1wire
sunxi-cir   - ИК
xradio_wlan

gpio 3 5 19 21 23 24 27
